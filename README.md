# Selenium Test Automation Boilerplate

Ready-to-use UI Test Automation Architecture using Java and Selenium WebDriver.

The project uses the following:
- **Java 11**
- **Selenium WebDriver**
- **WebDriverManager**
- **Univocity Parsers**
- **TestNG**
- **AssertJ**
- **Owner**
- **Extent Reports**
- **Selenium Shutterbug**
- **Gradle**
- **IntelliJ IDEA**
